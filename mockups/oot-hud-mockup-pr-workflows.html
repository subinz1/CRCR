Fix system Eigen 5.0.x detection in CMake #188793

Commit:
GitHub
Fixes detection of system-installed Eigen 5.0.x in CMake's find_package. The existing FindEigen3.cmake module only checked for Eigen3Config.cmake, missing the case where Eigen 5.0+ installs as Eigen3 with a different version scheme. Tested on Ubuntu 24.04 with Eigen 5.0.1 installed via apt. Pull Request resolved: https://github.com/pytorch/pytorch/pull/188793
